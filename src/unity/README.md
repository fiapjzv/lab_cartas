# Unity Client

Esta pasta contém o projeto **Unity** do jogo.

Aqui ficam todos os elementos responsáveis pela apresentação do jogo no cliente, incluindo lógica animação,
renderização, interação com o jogador e integração com o servidor.

> [!WARNING]  
> O código aqui de **NÃO DEVE CONTER REGRAS DE JOGO**, que devem viver no `core`.

## Responsabilidades

- Interface visual e renderização
- Input do jogador
- Animações e efeitos visuais
